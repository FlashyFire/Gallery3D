# Image Gallery 3D

Прототип 3D вьюера изображений.

HTML/CSS, Java Script, WebGL, three.js - Java Script 3D Library

Навигация по изображениям галлереи щелчком левой клавиши мыши по кнопкам "<" и ">".

Ближе/дальше - вращением колесика мыши.
